# Integração IPTV + SendPulse WhatsApp
Webhook para integração entre o painel IPTV e o SendPulse WhatsApp.
## Funcionalidades
- Criação automática de usuários de teste no painel IPTV
- Extração automática dos 4 últimos dígitos do número de telefone
- Envio das credenciais via WhatsApp usando a API do SendPulse
## Configuração
Defina as seguintes variáveis de ambiente:
- `SENDPULSE_CLIENT_ID`: ID do cliente do SendPulse
- `SENDPULSE_CLIENT_SECRET`: Secret do cliente do SendPulse
- `IPTV_API_URL`: URL da API do painel IPTV (padrão: https://mcapi.knewcms.com:
2087/lines/test)
